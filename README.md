# 有意美甲后端 
项目帮别人做的，时间比较赶，表也是按照他们课程的内容来做的，很多的设计有些不合理，希望后来者可以加以修改精进
集成了京东物流的 API，因为时间原因，没有做对应的数据封装和数据库表，做了一个假的 API 接口来模拟
聊天系统可以作为项目的亮点之一，主要使用 webSocket API 来实现
